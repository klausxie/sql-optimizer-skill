# Changelog

## [1.0.0] - 2026-03-18

### Added
- Initial version of SQL Optimizer data contracts
- 15 JSON Schema files:
  - sqlunit.schema.json - SQL unit definition
  - fragment_record.schema.json - SQL fragment records
  - optimization_proposal.schema.json - Optimization proposals
  - acceptance_result.schema.json - Validation acceptance results
  - patch_result.schema.json - Patch generation results
  - run_report.schema.json - Run report format
  - run_index.schema.json - Run index
  - verification_record.schema.json - Verification records
  - verification_summary.schema.json - Verification summaries
  - ops_health.schema.json - Operations health
  - ops_topology.schema.json - Operations topology
  - sql_artifact_index_row.schema.json - SQL artifact index
  - llm_candidate.schema.json - LLM candidate suggestions
  - llm_trace.schema.json - LLM trace data
  - policy.schema.json - Policy definitions
