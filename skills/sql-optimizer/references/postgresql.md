- V1 仅支持 `db.platform=postgresql`。
- validate 采用严格三态：`PASS/FAIL/NEED_MORE_PARAMS`。
- LLM provider 对照：
  - `opencode_run`：调用外部 `opencode run`，传输/解析失败立即失败。
  - `direct_openai_compatible`：调用 OpenAI-compatible `/chat/completions`，传输/解析失败立即失败。
  - `opencode_builtin`：本地确定性候选生成。
  - `heuristic`：最简本地启发式策略。
- 需要严格在线 LLM 生成时，优先使用 `direct_openai_compatible` 或 `opencode_run`。
- 需要 DB 感知提示时，配置：
  - `db.dsn`
  - `db.schema`（可选）
  - `db.statement_timeout_ms`
  - `db.allow_explain_analyze`（默认 false）
