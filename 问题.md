## 五、目录结构调整计划

### 5.1 当前结构 → 目标结构

```
当前:                              目标 (V8 设计):
runs/<run_id>/                     runs/<run_id>/
├── pipeline/                      ├── supervisor/        # 运行状态
│   ├── supervisor/                ├── project_context/   # 全局上下文
│   ├── scan/                      ├── sqlmap_catalog/    # SQL 目录
│   ├── optimize/                  │   ├── index.json
│   ├── validate/                  │   └── <sql_key>.json
│   ├── apply/                     ├── discovery/
│   ├── ops/                       ├── branching/
│   └── verification/              ├── pruning/
├── overview/                      ├── baseline/
├── sql/                           ├── optimize/
└── diagnostics/                   ├── validate/
                                   └── patch/
```

### 5.2 新增 .sqlopt/ 目录

```
.sqlopt/
├── cache/
│   ├── db_schemas/
│   │   └── {db_hash}/
│   │       ├── _meta.json
│   │       └── tables/
│   └── sqlmap_cache/
└── history/
    └── <archived_run_id>/
```
