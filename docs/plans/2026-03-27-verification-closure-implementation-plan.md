# Verification Closure Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the remaining correctness gaps in patch verification so applicable patches are proven against the generated artifact, not just the persisted `patchTarget` intent.

**Architecture:** First harden `patch_artifact` into an explicit verification base that can deterministically materialize supported unified diffs or emit precise artifact failures. Then route replay and syntax checks through that base for both statement and fragment paths, and finally lock the main verification failure matrix in focused regression tests. This work intentionally avoids patch-family expansion and report-layer refactors.

**Tech Stack:** Python 3.9, unittest/pytest, existing patch verification modules, unified diff generation in `patching_render`, current run-artifact pipeline

---

## File Map

- `python/sqlopt/verification/patch_artifact.py`
  Owns artifact target-path validation, supported unified-diff application, and artifact failure classification.
- `python/sqlopt/verification/patch_replay.py`
  Owns artifact-backed statement/fragment replay and target-SQL comparison.
- `python/sqlopt/verification/patch_syntax.py`
  Owns contract-driven syntax evidence derived from artifact-backed replay output.
- `python/sqlopt/stages/patch_generate.py`
  Orchestrates artifact materialization, replay, and syntax proof without embedding duplicate verification logic.
- `tests/test_patch_generate_orchestration.py`
  Locks end-to-end patch selection degradation when proof does not close.
- `tests/test_patch_replay.py`
  Locks statement/fragment replay behavior, drift reasons, and fragment-key resolution.
- `tests/test_patch_syntax.py`
  Locks syntax-evidence behavior under `dialectSyntaxCheckRequired=true/false`.
- `tests/test_verification_stage_integration.py`
  Locks the stage-level verification verdict when patch generation emits artifact-backed evidence.
